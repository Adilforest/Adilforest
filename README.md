### Featured project

🧾 **[MonServ — Receipt Storage, Analysis & Price Monitoring](https://github.com/Adilforest/monserv-portfolio)**
A polyglot (Go + Python) microservice system that ingests Kazakhstan fiscal receipts via manual entry,
OCR and OFD-URL parsing, normalises bilingual (RU/KK) product names, clusters them with transformer
embeddings + pgvector, and surfaces spending analytics in a React SPA. Event-driven over NATS JetStream,
contracts-first with Protobuf, delivered via Helm + ArgoCD (GitOps), with full Loki/Grafana observability.
